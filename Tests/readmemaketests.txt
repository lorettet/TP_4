Comment créer des tests:
lancer script setup.sh pour créer dossiers et fichiers run et description
lancer makemaketests.sh pour simuler les tests et sauvegarder les résultats
lancer renameoutputs pour mettre un .outfile à la fin des .dot crées
modifier le path pour arriver à anonyme.log dans changepath.sh et le lancer si besoin.

NB: finalement, le script qu’ils nous ont donné (test.sh) efface automatiquement les fichiers qu’il crée quand il les compare au .outfile
